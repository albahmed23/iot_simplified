# Open Project: Door Open Notification

An open project is where we post open sourced project using the latest in tech to build IoT devices and kits. You can follow us and learn how you can advance your skills in IoT.

## About

This project will show you how you can build a door open notification. The module we wil be building combines both 433 MHz Transmitter and Receiver to send a desktop notification to your PC letting you know that someone has either opened your door or is knocking on your door.

Please follow the step by step procedure on our website: IoTSimplified.com
